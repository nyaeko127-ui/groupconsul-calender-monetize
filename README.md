# グルコン開催管理システム

講師と運営が協力してグルコン（グルメコンシェルジュ）の開催日時を管理するシステムです。

## 機能

- **講師機能**
  - カレンダー上で候補日時を入力・提出
  - 提出済み候補の一覧表示とステータス確認

- **運営機能**
  - 講師から提出された候補日時の確認
  - 候補日時の確定・却下
  - 確定日時の一覧・カレンダー表示

## 技術スタック

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Zustand (状態管理)
- react-big-calendar (カレンダー表示)

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 使用方法

### ログイン

1. アプリケーションにアクセスすると、ログイン画面が表示されます
2. 名前、メールアドレス、ロール（講師または運営）を入力してログイン

### 講師として使用する場合

1. ログイン後、講師ダッシュボードが表示されます
2. カレンダー上で日付をクリックして候補日時を入力
3. 開始時刻と終了時刻を入力して提出
4. 提出済み候補の一覧でステータス（審査中/確定/却下）を確認

### 運営として使用する場合

1. ログイン後、運営ダッシュボードが表示されます
2. カレンダーで全ての候補日時を確認
3. 審査中候補の一覧から、各候補を確定または却下
4. 「確定日時一覧」ページで確定済みの日時を確認

## データ保存

現在のバージョンでは、データはブラウザのLocalStorageに保存されます。
ページをリロードしてもデータは保持されます。
